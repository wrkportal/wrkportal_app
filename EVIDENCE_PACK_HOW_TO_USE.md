## ğŸ”§ Troubleshooting

### **"Save as PDF" button doesn't work:**
- Just press **Ctrl+P** manually
- It does the same thing

### **PDF shows sidebar:**
- It shouldn't! The evidence pack page has no sidebar
- The action bar is hidden when printing
- If you see it, make sure you're on the evidence pack page

### **Want different filename:**
- When the print dialog opens
- Look for "Save as PDF"
- Click "Save"
- Type your custom filename

### **PDF looks wrong:**
- Make sure you selected "Portrait" orientation
- Make sure "Background graphics" is enabled
- Try different browser (Chrome works best)
