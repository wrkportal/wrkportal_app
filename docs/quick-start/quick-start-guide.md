# Quick Start Guide

## 5-Minute Setup

### Step 1: Create Your First Data Source

1. Go to **Reporting Studio** â†’ **Data Sources**
2. Click **"Add Data Source"**
3. Choose **"Upload File"**
4. Upload a CSV or Excel file
5. Click **"Save"**

### Step 2: Create a Dataset

1. Go to **Datasets**
2. Click **"New Dataset"**
3. Select your data source
4. Preview the data
5. Click **"Save Dataset"**

### Step 3: Create a Visualization

1. Go to **Visualizations**
2. Click **"New Visualization"**
3. Select your dataset
4. Choose **"Bar Chart"**
5. Select X and Y axes
6. Click **"Save"**

### Step 4: Build a Dashboard

1. Go to **Dashboards**
2. Click **"New Dashboard"**
3. Add your visualization
4. Arrange and customize
5. Click **"Save"**

## Common Workflows

### Workflow 1: Daily Sales Report

1. **Connect Data Source**: Connect to your sales database
2. **Create Dataset**: Query sales data for today
3. **Build Visualization**: Create a bar chart of sales by product
4. **Create Dashboard**: Combine multiple sales charts
5. **Schedule Delivery**: Set up daily email delivery

### Workflow 2: Weekly Team Dashboard

1. **Browse Marketplace**: Find a team dashboard template
2. **Install Template**: Install the template
3. **Customize**: Update with your team's data sources
4. **Share**: Share with team members
5. **Schedule**: Set up weekly delivery

### Workflow 3: Data Analysis

1. **Create Transformation**: Clean and filter your data
2. **Preview Results**: See transformed data
3. **Create Visualization**: Visualize the transformed data
4. **Export**: Export results to Excel or PDF
